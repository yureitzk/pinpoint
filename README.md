# Pinpoint

A simple game to test your spatial memory and precision by recreating point patterns on a canvas.

[Live website](https://yureitzk.github.io/pinpoint)

## Features

- Recreate point patterns by clicking on the canvas
- Adjustable difficulty settings
- Modes:
  - Memory - hides the target after a short time
  - Mirror - feflects the pattern across the divider
  - Absolute - recreate the pattern using exact point placement.
- Drag and drop text files with usernames for easy import
- Keyboard shortcuts for faster gameplay

## Requirements

Running and building requires `npm` and `Node.js`.

## Building

```sh
npm install

npm run build

# If you want to run a debug version
npm run dev
```
